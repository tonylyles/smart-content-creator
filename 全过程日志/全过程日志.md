## 14:02 - 系统联调启动

**问题**：UI 启动过程中遇到多个阻塞问题

**修复链**：
1. `ContentGenerator` 导入失败 → 修复 `src/generator/__init__.py`（包目录遮蔽 generator.py），用 importlib 动态加载
2. `data_storage.py` SyntaxError → 4个文件存在多重 UTF-8 BOM（EF BB BF ×3），清理 `data_storage.py`、`data_cleaner.py`、`knowledge_base.py`、`workflow.py`
3. Qdrant 未启动 → 启动 Docker Desktop + `docker compose up -d qdrant`
4. 模型名错误 → `config.py` 中 `gpt-4` 改为 `deepseek-v4-flash`（DeepSeek API 不接受 gpt-4）
5. Gradio 6 兼容性 → `launch()` 需要 `prevent_thread_lock=True`；`gr.Blocks(theme=)` 参数需移除
6. 旧进程端口占用 → 清理 7860 端口僵尸进程

**结果**：全链路测试成功，Pipeline 耗时 29.2 秒

## 14:58 - 发布计划页 KeyError 修复

**问题**：点击"已注册任务"按钮导致 UI 崩溃

**原因**：`get_stats()` 返回的 `pipeline` 字典中没有 `sla_target_sec` 键

**修复**：将 `stats['pipeline']['sla_target_sec']` 改为 `pipeline.get('sla_target_sec', 'N/A')`，所有字典取值改为 `.get()` 安全方式
